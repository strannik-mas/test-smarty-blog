# 📘 Тестовое задание (PHP + Smarty + MySQL)

Простое блог-приложение с категориями, статьями, сортировкой и пагинацией.
Реализовано с использованием PHP, MySQL, Smarty и Docker.

---

## 🚀 Функционал

* Главная страница со списком категорий
* Страница категории:

    * список статей
    * пагинация
    * сортировка (newest / oldest / popular)
* Страница статьи:

    * полный текст
    * изображение
    * счётчик просмотров
* Обработка 404 страниц
* ЧПУ URL:

    * `/`
    * `/category/{slug}`
    * `/article/{slug}`

---

## 🏗 Архитектура

Проект построен по принципам MVC:

* **Controllers** — обработка HTTP-запросов
* **Services** — бизнес-логика
* **Repositories** — работа с базой данных (PDO)
* **Mappers** — подготовка данных для шаблонов
* **Views (Smarty)** — отображение

## 🐳 Запуск проекта (Docker)

### 1. Клонировать репозиторий

```bash
git clone <repo-url>
cd test-smarty
```

---

### 2. Запустить контейнеры

```bash
docker compose up -d --build
```

---

### 3. Открыть в браузере

```text
http://localhost:8080
```

---

## 🗄 База данных

Для создания структуры базы данных и заполнения тестовыми данными используется CLI-скрипт.

### Быстрый запуск (рекомендуется)

```bash
docker compose exec app php cli.php migrate
docker compose exec app php cli.php seed
```

---

### Альтернативный способ (через shell контейнера)

```bash
docker exec -it test_smarty_app bash
php cli.php migrate
php cli.php seed
```

---

### Что делают команды

* `migrate` — создаёт таблицы
* `seed` — заполняет тестовыми данными

---

### Важно

Команды выполняются **внутри контейнера**, так как приложение и база данных настроены в Docker-окружении.


Структура:

* `categories`
* `articles`
* `article_category`
